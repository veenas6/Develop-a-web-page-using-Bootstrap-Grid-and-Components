## 🚀 Project Overview

This project is a **fully responsive website** developed using **Bootstrap 5**.
It demonstrates the use of Bootstrap’s **Grid System**, **Components**, and **Responsive Layout** to create a modern web interface.

The website includes:

* Navigation Bar
* Hero Section
* Services Cards
* Modal Popup
* Footer Section

---

## ✨ Features

✅ Responsive Navbar with toggle menu
✅ Hero section with gradient background
✅ Bootstrap Grid layout
✅ Service cards with shadow effects
✅ Interactive modal popup
✅ Mobile-friendly design
✅ Clean UI using Bootstrap components
